# Angry Birds Mini

A single-page HTML + JavaScript mini game inspired by Angry Birds.

## Features
- Drag-and-release slingshot gameplay
- Simple physics and destructible blocks
- Score and remaining shots tracking
- Works as a static GitHub Pages site

## Run locally
Open `index.html` in a browser, or serve it:

```bash
python3 -m http.server 8080
```

Then visit `http://127.0.0.1:8080`
